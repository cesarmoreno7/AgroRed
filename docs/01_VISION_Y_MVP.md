# Vision y MVP inicial

## Vision de negocio

Segun el plan de negocio, AGRORED debe operar como una infraestructura digital para gestionar el sistema alimentario territorial y no solo como una aplicacion transaccional aislada. El valor central para el municipio es:

- integrar actores rurales, institucionales, logisticos y comerciales
- reducir desperdicio alimentario
- mejorar la planeacion de programas alimentarios
- generar inteligencia territorial para decisiones publicas

## Actores del ecosistema

- administracion municipal
- productor rural
- supermercado o comercio aliado
- comedor comunitario o programa institucional
- operador logistico
- analista territorial

## Corte funcional recomendado para el MVP

### Entra al MVP

- autenticacion y roles
- registro de municipios y actores
- registro de productores
- publicacion de oferta alimentaria
- registro de demanda institucional
- excedentes alimentarios y rescate
- tablero operacional basico
- trazabilidad minima de entregas

### Se difiere a la siguiente fase

- observatorio nacional completo
- IRAT automatizado
- modelos de machine learning
- automatizacion con Airflow
- Power BI embebido
- app movil Flutter

## Principios del producto

- multiactor desde el inicio
- enfoque municipal primero
- trazabilidad y auditoria por defecto
- georreferenciacion preparada desde la base de datos
- modularidad por microservicios
